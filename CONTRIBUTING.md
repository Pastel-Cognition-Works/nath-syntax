# Contributing to NATH Syntax

Thank you for your interest in improving the NATH syntax extension!

This repository focuses exclusively on:

- syntax highlighting  
- grammar definitions  
- snippets  
- themes  
- documentation  
- example files  

If you would like to contribute, please follow the guidelines below.
